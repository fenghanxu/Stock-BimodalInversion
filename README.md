# Stock-BimodalInversion

说明:  该战法需要寻找两个相近山峰,画一条一线,如果出现突破这条直线就买入, 或者寻找两个相近的低谷,画一条一线,如果出现突破这条直线就买入.

目前代码还是停留寻找单个山峰里面,单个山峰找出来很不准,代码也比较难维护,山峰寻找比较准的可以参考([Stock-KLine-Breakthrough](https://github.com/fenghanxu/Stock-KLine-Breakthrough))
